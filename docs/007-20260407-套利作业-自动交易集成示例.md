# 套利作业自动交易集成示例

## 概述

本文档提供了如何在 `FlinkADSArbitrageJob` 中集成自动交易功能的完整示例代码。

## 核心代码示例

### 1. 添加交易处理函数

在 `FlinkADSArbitrageJob.java` 中添加以下内部类：

```java
/**
 * 套利交易处理器
 * 
 * 功能：
 * 1. 管理持仓状态
 * 2. 判断开仓/平仓条件
 * 3. 调用交易服务下单
 * 4. 记录交易日志
 */
public static class ArbitrageTrader extends KeyedProcessFunction<String, ArbitrageOpportunity, TradeRecord> {
    
    // 配置参数
    private final boolean tradingEnabled;
    private final BigDecimal tradeAmount;
    private final int maxPositions;
    private final BigDecimal openThreshold;
    private final BigDecimal closeThreshold;
    private final long maxHoldTimeMs;
    
    // 交易服务
    private transient OKXTradingService tradingService;
    
    // 持仓状态
    private transient ValueState<PositionState> positionState;
    
    // 全局持仓计数器（使用 MapState 模拟）
    private transient MapState<String, Integer> globalPositionCount;
    
    public ArbitrageTrader(ConfigLoader config) {
        this.tradingEnabled = config.getBoolean("arbitrage.trading.enabled", false);
        this.tradeAmount = new BigDecimal(config.getString("arbitrage.trading.trade-amount", "100"));
        this.maxPositions = config.getInt("arbitrage.trading.max-positions", 5);
        this.openThreshold = new BigDecimal(config.getString("arbitrage.trading.open-threshold", "0.005"));
        this.closeThreshold = new BigDecimal(config.getString("arbitrage.trading.close-threshold", "0.002"));
        this.maxHoldTimeMs = config.getLong("arbitrage.trading.max-hold-time-minutes", 60L) * 60 * 1000;
    }
    
    @Override
    public void open(Configuration parameters) {
        // 初始化交易服务
        ConfigLoader config = ConfigLoader.getInstance();
        tradingService = new OKXTradingService(config);
        
        if (!tradingService.isConfigured()) {
            logger.warn("⚠ OKX API 未配置，自动交易功能将被禁用");
        } else if (tradingEnabled) {
            logger.info("✓ 自动交易功能已启用（{}模式）", 
                tradingService.isSimulated() ? "模拟" : "实盘");
        } else {
            logger.info("ℹ 自动交易功能已禁用");
        }
        
        // 初始化状态
        ValueStateDescriptor<PositionState> positionDescriptor = new ValueStateDescriptor<>(
            "position-state",
            PositionState.class
        );
        positionState = getRuntimeContext().getState(positionDescriptor);
        
        MapStateDescriptor<String, Integer> countDescriptor = new MapStateDescriptor<>(
            "global-position-count",
            String.class,
            Integer.class
        );
        globalPositionCount = getRuntimeContext().getMapState(countDescriptor);
    }
    
    @Override
    public void processElement(
            ArbitrageOpportunity opportunity,
            Context ctx,
            Collector<TradeRecord> out) throws Exception {
        
        // 检查是否启用交易
        if (!tradingEnabled || !tradingService.isConfigured()) {
            return;
        }
        
        // 读取当前持仓状态
        PositionState position = positionState.value();
        
        if (position == null || !position.isOpen()) {
            // 没有持仓，判断是否开仓
            tryOpenPosition(opportunity, out);
        } else {
            // 有持仓，判断是否平仓
            tryClosePosition(opportunity, position, out);
        }
    }
    
    /**
     * 尝试开仓
     */
    private void tryOpenPosition(ArbitrageOpportunity opportunity, Collector<TradeRecord> out) throws Exception {
        // 检查是否达到最大持仓数量
        int currentPositions = getCurrentPositionCount();
        if (currentPositions >= maxPositions) {
            logger.debug("已达到最大持仓数量: {}/{}", currentPositions, maxPositions);
            return;
        }
        
        // 检查价差率是否超过开仓阈值
        BigDecimal spreadRate = opportunity.spreadRate.divide(new BigDecimal("100"), 6, RoundingMode.HALF_UP);
        if (spreadRate.abs().compareTo(openThreshold) <= 0) {
            return;
        }
        
        logger.info("🎯 发现套利机会，准备开仓: {} | 价差率: {}% | 方向: {}",
            opportunity.symbol, opportunity.spreadRate, opportunity.arbitrageDirection);
        
        // 创建持仓状态
        PositionState position = new PositionState();
        position.setSymbol(opportunity.symbol);
        position.setAmount(tradeAmount);
        position.setEntrySpreadRate(opportunity.spreadRate);
        position.setEntrySpotPrice(opportunity.spotPrice);
        position.setEntrySwapPrice(opportunity.futuresPrice);
        position.setOpenTime(System.currentTimeMillis());
        position.setOpen(true);
        
        // 根据套利方向下单
        boolean success = false;
        if (opportunity.arbitrageDirection.contains("做多现货")) {
            // 策略 A：做多现货 + 做空合约
            position.setDirection("LONG_SPOT_SHORT_SWAP");
            String spotOrderId = tradingService.buySpot(opportunity.symbol, tradeAmount);
            String swapOrderId = tradingService.shortSwap(opportunity.symbol, tradeAmount, 1);
            
            if (spotOrderId != null && swapOrderId != null) {
                position.setSpotOrderId(spotOrderId);
                position.setSwapOrderId(swapOrderId);
                success = true;
            }
        } else {
            // 策略 B：做空现货 + 做多合约
            position.setDirection("SHORT_SPOT_LONG_SWAP");
            String spotOrderId = tradingService.sellSpot(opportunity.symbol, tradeAmount);
            String swapOrderId = tradingService.longSwap(opportunity.symbol, tradeAmount, 1);
            
            if (spotOrderId != null && swapOrderId != null) {
                position.setSpotOrderId(spotOrderId);
                position.setSwapOrderId(swapOrderId);
                success = true;
            }
        }
        
        if (success) {
            // 保存持仓状态
            positionState.update(position);
            incrementPositionCount();
            
            logger.info("✅ 开仓成功: {} | 方向: {} | 金额: {} USDT | 现货订单: {} | 合约订单: {}",
                position.getSymbol(), position.getDirection(), position.getAmount(),
                position.getSpotOrderId(), position.getSwapOrderId());
            
            // 输出交易记录
            TradeRecord record = new TradeRecord();
            record.symbol = opportunity.symbol;
            record.action = "OPEN";
            record.direction = position.getDirection();
            record.amount = tradeAmount;
            record.spreadRate = opportunity.spreadRate;
            record.spotPrice = opportunity.spotPrice;
            record.swapPrice = opportunity.futuresPrice;
            record.timestamp = System.currentTimeMillis();
            out.collect(record);
        } else {
            logger.error("❌ 开仓失败: {}", opportunity.symbol);
        }
    }
    
    /**
     * 尝试平仓
     */
    private void tryClosePosition(
            ArbitrageOpportunity opportunity,
            PositionState position,
            Collector<TradeRecord> out) throws Exception {
        
        // 检查平仓条件
        boolean shouldClose = false;
        String closeReason = "";
        
        // 条件 1：价差率回归到平仓阈值以下
        BigDecimal currentSpreadRate = opportunity.spreadRate.divide(new BigDecimal("100"), 6, RoundingMode.HALF_UP);
        if (currentSpreadRate.abs().compareTo(closeThreshold) <= 0) {
            shouldClose = true;
            closeReason = "价差率回归";
        }
        
        // 条件 2：持仓时间超过最大持仓时间
        long holdTime = System.currentTimeMillis() - position.getOpenTime();
        if (holdTime > maxHoldTimeMs) {
            shouldClose = true;
            closeReason = "超时平仓";
        }
        
        // 条件 3：价差反向（亏损）
        BigDecimal entrySpreadRate = position.getEntrySpreadRate().divide(new BigDecimal("100"), 6, RoundingMode.HALF_UP);
        if (entrySpreadRate.multiply(currentSpreadRate).compareTo(BigDecimal.ZERO) < 0) {
            shouldClose = true;
            closeReason = "价差反向";
        }
        
        if (!shouldClose) {
            return;
        }
        
        logger.info("🔄 准备平仓: {} | 原因: {} | 持仓时间: {} 分钟",
            position.getSymbol(), closeReason, holdTime / 60000);
        
        // 根据持仓方向平仓
        boolean success = false;
        if ("LONG_SPOT_SHORT_SWAP".equals(position.getDirection())) {
            // 平仓：卖出现货 + 平空仓
            String spotOrderId = tradingService.sellSpot(position.getSymbol(), position.getAmount());
            String swapOrderId = tradingService.closeShortSwap(position.getSymbol(), position.getAmount());
            success = (spotOrderId != null && swapOrderId != null);
        } else {
            // 平仓：买入现货 + 平多仓
            String spotOrderId = tradingService.buySpot(position.getSymbol(), position.getAmount());
            String swapOrderId = tradingService.closeLongSwap(position.getSymbol(), position.getAmount());
            success = (spotOrderId != null && swapOrderId != null);
        }
        
        if (success) {
            // 计算盈亏
            BigDecimal profit = calculateProfit(position, opportunity);
            
            logger.info("✅ 平仓成功: {} | 原因: {} | 盈亏: {} USDT",
                position.getSymbol(), closeReason, profit);
            
            // 清空持仓状态
            position.setOpen(false);
            positionState.update(position);
            decrementPositionCount();
            
            // 输出交易记录
            TradeRecord record = new TradeRecord();
            record.symbol = position.getSymbol();
            record.action = "CLOSE";
            record.direction = position.getDirection();
            record.amount = position.getAmount();
            record.spreadRate = opportunity.spreadRate;
            record.spotPrice = opportunity.spotPrice;
            record.swapPrice = opportunity.futuresPrice;
            record.profit = profit;
            record.closeReason = closeReason;
            record.holdTimeMs = holdTime;
            record.timestamp = System.currentTimeMillis();
            out.collect(record);
        } else {
            logger.error("❌ 平仓失败: {}", position.getSymbol());
        }
    }
    
    /**
     * 计算盈亏
     */
    private BigDecimal calculateProfit(PositionState position, ArbitrageOpportunity current) {
        // 简化计算：(平仓价差 - 开仓价差) * 金额
        BigDecimal entrySpread = position.getEntrySwapPrice().subtract(position.getEntrySpotPrice());
        BigDecimal currentSpread = current.futuresPrice.subtract(current.spotPrice);
        
        BigDecimal spreadDiff = currentSpread.subtract(entrySpread);
        
        // 根据方向调整盈亏符号
        if ("SHORT_SPOT_LONG_SWAP".equals(position.getDirection())) {
            spreadDiff = spreadDiff.negate();
        }
        
        // 扣除手续费（假设 0.1%）
        BigDecimal fee = position.getAmount().multiply(new BigDecimal("0.002"));  // 双边手续费
        
        return spreadDiff.subtract(fee);
    }
    
    /**
     * 获取当前持仓数量
     */
    private int getCurrentPositionCount() throws Exception {
        Integer count = globalPositionCount.get("count");
        return count != null ? count : 0;
    }
    
    /**
     * 增加持仓计数
     */
    private void incrementPositionCount() throws Exception {
        int count = getCurrentPositionCount();
        globalPositionCount.put("count", count + 1);
    }
    
    /**
     * 减少持仓计数
     */
    private void decrementPositionCount() throws Exception {
        int count = getCurrentPositionCount();
        globalPositionCount.put("count", Math.max(0, count - 1));
    }
}

/**
 * 交易记录
 */
public static class TradeRecord {
    public String symbol;
    public String action;  // OPEN / CLOSE
    public String direction;
    public BigDecimal amount;
    public BigDecimal spreadRate;
    public BigDecimal spotPrice;
    public BigDecimal swapPrice;
    public BigDecimal profit;
    public String closeReason;
    public long holdTimeMs;
    public long timestamp;
}
```

### 2. 集成到主流程

在 `main()` 方法中，在黑名单过滤之后添加交易处理：

```java
// ========== 步骤 6: 添加自动交易处理 ==========
logger.info("配置自动交易处理...");

DataStream<TradeRecord> tradeStream = filteredStream
    .keyBy(opportunity -> opportunity.symbol)
    .process(new ArbitrageTrader(config))
    .name("Arbitrage Trader");

logger.info("✓ 自动交易处理配置成功");

// ========== 步骤 7: 转换为 JSON 并写入 Doris ==========
// 原有的套利机会写入 Doris
DataStream<String> opportunityJsonStream = filteredStream
    .map(opportunity -> {
        // ... 原有代码 ...
    })
    .name("Opportunity To JSON");

opportunityJsonStream.sinkTo(opportunitySink).name("Doris Opportunity Sink");

// 交易记录写入 Doris
DataStream<String> tradeJsonStream = tradeStream
    .map(trade -> {
        ObjectNode json = OBJECT_MAPPER.createObjectNode();
        json.put("symbol", trade.symbol);
        json.put("action", trade.action);
        json.put("direction", trade.direction);
        json.put("amount", trade.amount.toString());
        json.put("spread_rate", trade.spreadRate.toString());
        json.put("spot_price", trade.spotPrice.toString());
        json.put("swap_price", trade.swapPrice.toString());
        if (trade.profit != null) {
            json.put("profit", trade.profit.toString());
        }
        if (trade.closeReason != null) {
            json.put("close_reason", trade.closeReason);
        }
        if (trade.holdTimeMs > 0) {
            json.put("hold_time_minutes", trade.holdTimeMs / 60000);
        }
        json.put("timestamp", trade.timestamp);
        return json.toString();
    })
    .name("Trade To JSON");

DorisSink<String> tradeSink = dorisSinkFactory.createDorisSink(
    "crypto_dw",
    "ads_arbitrage_trades",
    "ads-arbitrage-trades"
);

tradeJsonStream.sinkTo(tradeSink).name("Doris Trade Sink");

logger.info("✓ Doris Trade Sink 创建成功");
logger.info("  Database: crypto_dw");
logger.info("  Table: ads_arbitrage_trades");
```

### 3. 配置文件

```yaml
# 套利交易配置
arbitrage:
  trading:
    enabled: false                   # 默认禁用，需要手动启用
    trade-amount: 100                # 单笔交易金额（USDT）
    max-positions: 5                 # 最大持仓数量

  api:
    key: ${OKX_API_KEY:}
    secret: ${OKX_SECRET_KEY:}
    passphrase: ${OKX_PASSPHRASE:}
    simulated: true                  # 默认使用模拟交易
```

### 4. 创建 Doris 表

```sql
-- 交易记录表
CREATE TABLE IF NOT EXISTS crypto_dw.ads_arbitrage_trades (
    symbol VARCHAR(20) COMMENT '交易对',
    action VARCHAR(10) COMMENT '操作类型：OPEN/CLOSE',
    direction VARCHAR(30) COMMENT '套利方向',
    amount DECIMAL(20, 8) COMMENT '交易金额',
    spread_rate DECIMAL(10, 6) COMMENT '价差率（%）',
    spot_price DECIMAL(20, 8) COMMENT '现货价格',
    swap_price DECIMAL(20, 8) COMMENT '合约价格',
    profit DECIMAL(20, 8) COMMENT '盈亏',
    close_reason VARCHAR(50) COMMENT '平仓原因',
    hold_time_minutes BIGINT COMMENT '持仓时间（分钟）',
    timestamp BIGINT COMMENT '时间戳'
)
DUPLICATE KEY(symbol, timestamp)
DISTRIBUTED BY HASH(symbol) BUCKETS 10
PROPERTIES (
    "replication_num" = "1"
);
```

## 使用说明

### 1. 启用自动交易

```bash
# 1. 配置 API 密钥
export OKX_API_KEY="your-api-key"
export OKX_SECRET_KEY="your-secret-key"
export OKX_PASSPHRASE="your-passphrase"

# 2. 修改配置文件
vim data-warehouse/src/main/resources/config/application-dev.yml
# 设置 arbitrage.trading.enabled: true

# 3. 启动作业
cd data-warehouse
bash run-flink-ads-arbitrage.sh
```

### 2. 监控交易

```bash
# 查看交易日志
tail -f logs/flink-ads-arbitrage.log | grep "交易\|开仓\|平仓"

# 查看交易记录
mysql -h 127.0.0.1 -P 9030 -u root -e "
SELECT 
    symbol,
    action,
    direction,
    amount,
    spread_rate,
    profit,
    close_reason,
    hold_time_minutes,
    FROM_UNIXTIME(timestamp/1000) as trade_time
FROM crypto_dw.ads_arbitrage_trades
ORDER BY timestamp DESC
LIMIT 20
"
```

### 3. 统计分析

```sql
-- 总盈亏统计
SELECT 
    symbol,
    COUNT(*) as trade_count,
    SUM(CASE WHEN action='CLOSE' THEN profit ELSE 0 END) as total_profit,
    AVG(CASE WHEN action='CLOSE' THEN profit ELSE NULL END) as avg_profit,
    AVG(CASE WHEN action='CLOSE' THEN hold_time_minutes ELSE NULL END) as avg_hold_time
FROM crypto_dw.ads_arbitrage_trades
GROUP BY symbol
ORDER BY total_profit DESC;

-- 平仓原因分析
SELECT 
    close_reason,
    COUNT(*) as count,
    AVG(profit) as avg_profit
FROM crypto_dw.ads_arbitrage_trades
WHERE action = 'CLOSE'
GROUP BY close_reason;
```

## 注意事项

1. **风险提示**：自动交易有风险，建议先在模拟环境充分测试
2. **资金管理**：确保账户有足够的资金，建议预留 2-3 倍的交易金额
3. **监控告警**：建议配置监控告警，及时发现异常
4. **日志分析**：定期分析交易日志，优化策略参数
