# 套利作业订单流集成方案

## 概述

通过 WebSocket 订阅 OKX 订单更新，实现更可靠的套利交易流程：
1. 发现套利机会 → 下单
2. 订阅订单状态 → 等待成交
3. 现货和合约都成交 → 确认开仓
4. 套利机会消失 → 同时平仓

## 架构设计

### 数据流图

```
套利机会流 ──────────────┐
                         │
                         ├─→ 判断开仓条件
                         │       │
                         │       ├─ 下单（现货 + 合约）
                         │       │
                         │       └─ 注册待匹配订单
                         │
订单更新流（WebSocket）──┤
                         │
                         ├─→ 订单匹配处理器
                         │       │
                         │       ├─ 现货订单成交 ✓
                         │       ├─ 合约订单成交 ✓
                         │       │
                         │       └─ 两个都成交 → 确认开仓 → 保存持仓状态
                         │
套利机会流 ──────────────┤
                         │
                         └─→ 判断平仓条件
                                 │
                                 ├─ 价差回归
                                 │
                                 └─ 同时平仓（现货 + 合约）
```

### 核心组件

#### 1. OKXOrderWebSocketSource
- 订阅 OKX 私有频道的订单更新
- 实时接收订单状态变化
- 发送到 Flink 订单流

#### 2. OrderMatchProcessor
- 管理待匹配订单（MapState）
- 匹配现货和合约订单
- 确认开仓成功后输出持仓状态

#### 3. ArbitrageTraderV2（改进版）
- 发现套利机会时下单
- 注册待匹配订单到 OrderMatchProcessor
- 监控持仓状态
- 判断平仓条件并执行平仓

public static void main(String[] args) throws Exception {
    // ... 前面的代码 ...
    
    // ========== 步骤 6: 创建订单流 ==========
    logger.info("创建订单 WebSocket 流...");
    
    DataStream<String> orderStream = env
        .addSource(new OKXOrderWebSocketSource(config))
        .name("OKX Order WebSocket Source");
    
    logger.info("✓ 订单流创建成功");
    
    // ========== 步骤 7: 订单匹配处理 ==========
    logger.info("配置订单匹配处理器...");
    
    DataStream<PositionState> confirmedPositionStream = orderStream
        .keyBy(message -> {
            // 从订单消息中提取交易对作为 key
            try {
                JsonNode rootNode = OBJECT_MAPPER.readTree(message);
                if (rootNode.has("data") && rootNode.get("data").isArray()) {
                    JsonNode order = rootNode.get("data").get(0);
                    String instId = order.get("instId").asText();
                    return instId.replace("-SWAP", "");  // 统一为交易对
                }
            } catch (Exception e) {
                logger.error("解析订单消息失败: {}", e.getMessage());
            }
            return "UNKNOWN";
        })
        .process(new OrderMatchProcessor())
        .name("Order Match Processor");
    
    logger.info("✓ 订单匹配处理器配置成功");
    
    // ========== 步骤 8: 套利交易处理（改进版）==========
    logger.info("配置套利交易处理器...");
    
    // 将套利机会流和确认持仓流合并
    DataStream<Object> tradingInputStream = filteredStream
        .map(opp -> (Object) opp)
        .union(confirmedPositionStream.map(pos -> (Object) pos));
    
    DataStream<TradeRecord> tradeStream = tradingInputStream
        .keyBy(obj -> {
            if (obj instanceof ArbitrageOpportunity) {
                return ((ArbitrageOpportunity) obj).symbol;
            } else if (obj instanceof PositionState) {
                return ((PositionState) obj).getSymbol();
            }
            return "UNKNOWN";
        })
        .process(new ArbitrageTraderV2(config))
        .name("Arbitrage Trader V2");
    
    logger.info("✓ 套利交易处理器配置成功");

}

### 2. 改进的交易处理器

```java
/**
 * 套利交易处理器 V2
 * 
 * 改进：
 * 1. 下单后等待订单成交确认
 * 2. 只有现货和合约都成交才确认开仓
 * 3. 平仓时同时平掉现货和合约
 */
public static class ArbitrageTraderV2 extends KeyedProcessFunction<String, Object, TradeRecord> {
    
    private final boolean tradingEnabled;
    private final BigDecimal tradeAmount;
    private final BigDecimal openThreshold;
    private final BigDecimal closeThreshold;
    
    private transient OKXTradingService tradingService;
    private transient ValueState<PositionState> positionState;
    private transient ValueState<PendingOpenOrder> pendingOpenState;
    
    public ArbitrageTraderV2(ConfigLoader config) {
        this.tradingEnabled = config.getBoolean("arbitrage.trading.enabled", false);
        this.tradeAmount = new BigDecimal(config.getString("arbitrage.trading.trade-amount", "100"));
        this.openThreshold = new BigDecimal(config.getString("arbitrage.trading.open-threshold", "0.005"));
        this.closeThreshold = new BigDecimal(config.getString("arbitrage.trading.close-threshold", "0.002"));
    }
    
    @Override
    public void open(Configuration parameters) {
        // 初始化交易服务
        ConfigLoader config = ConfigLoader.getInstance();
        tradingService = new OKXTradingService(config);
        
        // 初始化持仓状态
        ValueStateDescriptor<PositionState> positionDescriptor = new ValueStateDescriptor<>(
            "position-state-v2",
            PositionState.class
        );
        positionState = getRuntimeContext().getState(positionDescriptor);
        
        // 初始化待开仓订单状态
        ValueStateDescriptor<PendingOpenOrder> pendingDescriptor = new ValueStateDescriptor<>(
            "pending-open-order",
            PendingOpenOrder.class
        );
        pendingOpenState = getRuntimeContext().getState(pendingDescriptor);
        
        logger.info("ArbitrageTraderV2 初始化完成");
    }
    
    @Override
    public void processElement(
            Object value,
            Context ctx,
            Collector<TradeRecord> out) throws Exception {
        
        if (!tradingEnabled || !tradingService.isConfigured()) {
            return;
        }
        
        if (value instanceof ArbitrageOpportunity) {
            // 处理套利机会
            processArbitrageOpportunity((ArbitrageOpportunity) value, ctx, out);
        } else if (value instanceof PositionState) {
            // 处理确认的持仓
            processConfirmedPosition((PositionState) value, out);
        }
    }
    
    /**
     * 处理套利机会
     */
    private void processArbitrageOpportunity(
            ArbitrageOpportunity opportunity,
            Context ctx,
            Collector<TradeRecord> out) throws Exception {
        
        PositionState position = positionState.value();
        PendingOpenOrder pending = pendingOpenState.value();
        
        // 情况 1：没有持仓，也没有待开仓订单 → 尝试开仓
        if ((position == null || !position.isOpen()) && pending == null) {
            tryOpenPosition(opportunity, out);
        }
        // 情况 2：有持仓 → 判断是否平仓
        else if (position != null && position.isOpen()) {
            tryClosePosition(opportunity, position, out);
        }
        // 情况 3：有待开仓订单 → 等待订单成交（不做任何操作）
    }
    
    /**
     * 处理确认的持仓
     */
    private void processConfirmedPosition(
            PositionState confirmedPosition,
            Collector<TradeRecord> out) throws Exception {
        
        // 保存持仓状态
        positionState.update(confirmedPosition);
        
        // 清空待开仓订单
        pendingOpenState.clear();
        
        logger.info("✅ 持仓已确认并保存: {}", confirmedPosition.getSymbol());
        
        // 输出开仓记录
        TradeRecord record = new TradeRecord();
        record.symbol = confirmedPosition.getSymbol();
        record.action = "OPEN_CONFIRMED";
        record.direction = confirmedPosition.getDirection();
        record.amount = confirmedPosition.getAmount();
        record.spreadRate = confirmedPosition.getEntrySpreadRate();
        record.spotPrice = confirmedPosition.getEntrySpotPrice();
        record.swapPrice = confirmedPosition.getEntrySwapPrice();
        record.timestamp = System.currentTimeMillis();
        out.collect(record);
    }
    
    /**
     * 尝试开仓
     */
    private void tryOpenPosition(
            ArbitrageOpportunity opportunity,
            Collector<TradeRecord> out) throws Exception {
        
        // 检查价差率
        BigDecimal spreadRate = opportunity.spreadRate.divide(new BigDecimal("100"), 6, java.math.RoundingMode.HALF_UP);
        if (spreadRate.abs().compareTo(openThreshold) <= 0) {
            return;
        }
        
        logger.info("🎯 发现套利机会，准备下单: {} | 价差率: {}%",
            opportunity.symbol, opportunity.spreadRate);
        
        // 下单
        String spotOrderId = null;
        String swapOrderId = null;
        String direction = null;
        
        if (opportunity.arbitrageDirection.contains("做多现货")) {
            // 策略 A：做多现货 + 做空合约
            direction = "LONG_SPOT_SHORT_SWAP";
            spotOrderId = tradingService.buySpot(opportunity.symbol, tradeAmount);
            swapOrderId = tradingService.shortSwap(opportunity.symbol, tradeAmount, 1);
        } else {
            // 策略 B：做空现货 + 做多合约
            direction = "SHORT_SPOT_LONG_SWAP";
            spotOrderId = tradingService.sellSpot(opportunity.symbol, tradeAmount);
            swapOrderId = tradingService.longSwap(opportunity.symbol, tradeAmount, 1);
        }
        
        if (spotOrderId != null && swapOrderId != null) {
            // 保存待开仓订单信息
            PendingOpenOrder pending = new PendingOpenOrder();
            pending.symbol = opportunity.symbol;
            pending.direction = direction;
            pending.spotOrderId = spotOrderId;
            pending.swapOrderId = swapOrderId;
            pending.amount = tradeAmount;
            pending.createTime = System.currentTimeMillis();
            pendingOpenState.update(pending);
            
            logger.info("📝 订单已提交，等待成交确认: spotOrderId={}, swapOrderId={}",
                spotOrderId, swapOrderId);
            
            // 输出下单记录
            TradeRecord record = new TradeRecord();
            record.symbol = opportunity.symbol;
            record.action = "OPEN_PENDING";
            record.direction = direction;
            record.amount = tradeAmount;
            record.spreadRate = opportunity.spreadRate;
            record.spotPrice = opportunity.spotPrice;
            record.swapPrice = opportunity.futuresPrice;
            record.timestamp = System.currentTimeMillis();
            out.collect(record);
        } else {
            logger.error("❌ 下单失败: {}", opportunity.symbol);
        }
    }
    
    /**
     * 尝试平仓
     */
    private void tryClosePosition(
            ArbitrageOpportunity opportunity,
            PositionState position,
            Collector<TradeRecord> out) throws Exception {
        
        // 检查平仓条件
        BigDecimal currentSpreadRate = opportunity.spreadRate.divide(new BigDecimal("100"), 6, java.math.RoundingMode.HALF_UP);
        
        if (currentSpreadRate.abs().compareTo(closeThreshold) > 0) {
            // 价差还在，不平仓
            return;
        }
        
        logger.info("🔄 价差回归，准备平仓: {} | 当前价差率: {}%",
            position.getSymbol(), opportunity.spreadRate);
        
        // 同时平仓
        boolean success = false;
        if ("LONG_SPOT_SHORT_SWAP".equals(position.getDirection())) {
            // 平仓：卖出现货 + 平空仓
            String spotOrderId = tradingService.sellSpot(position.getSymbol(), position.getAmount());
            String swapOrderId = tradingService.closeShortSwap(position.getSymbol(), position.getAmount());
            success = (spotOrderId != null && swapOrderId != null);
        } else {
            // 平仓：买入现货 + 平多仓
            String spotOrderId = tradingService.buySpot(position.getSymbol(), position.getAmount());
            String swapOrderId = tradingService.closeLongSwap(position.getSymbol(), position.getAmount());
            success = (spotOrderId != null && swapOrderId != null);
        }
        
        if (success) {
            // 计算盈亏
            BigDecimal entrySpread = position.getEntrySwapPrice().subtract(position.getEntrySpotPrice());
            BigDecimal currentSpread = opportunity.futuresPrice.subtract(opportunity.spotPrice);
            BigDecimal spreadDiff = currentSpread.subtract(entrySpread);
            
            if ("SHORT_SPOT_LONG_SWAP".equals(position.getDirection())) {
                spreadDiff = spreadDiff.negate();
            }
            
            BigDecimal fee = position.getAmount().multiply(new BigDecimal("0.002"));
            BigDecimal profit = spreadDiff.subtract(fee);
            
            long holdTime = System.currentTimeMillis() - position.getOpenTime();
            
            logger.info("✅ 平仓成功: {} | 盈亏: {} USDT | 持仓时间: {} 分钟",
                position.getSymbol(), profit, holdTime / 60000);
            
            // 清空持仓
            position.setOpen(false);
            positionState.update(position);
            
            // 输出平仓记录
            TradeRecord record = new TradeRecord();
            record.symbol = position.getSymbol();
            record.action = "CLOSE";
            record.direction = position.getDirection();
            record.amount = position.getAmount();
            record.spreadRate = opportunity.spreadRate;
            record.spotPrice = opportunity.spotPrice;
            record.swapPrice = opportunity.futuresPrice;
            record.profit = profit;
            record.closeReason = "价差回归";
            record.holdTimeMs = holdTime;
            record.timestamp = System.currentTimeMillis();
            out.collect(record);
        } else {
            logger.error("❌ 平仓失败: {}", position.getSymbol());
        }
    }
    
    /**
     * 待开仓订单
     */
    public static class PendingOpenOrder implements java.io.Serializable {
        private static final long serialVersionUID = 1L;
        
        public String symbol;
        public String direction;
        public String spotOrderId;
        public String swapOrderId;
        public BigDecimal amount;
        public long createTime;
    }
}
```

### 2. 更精确的成交价格
- ✅ 使用实际成交价格计算价差
- ✅ 更准确的盈亏计算

### 3. 更好的风险控制
- ✅ 订单超时处理
- ✅ 部分成交处理
- ✅ 异常情况告警

### 4. 同时平仓
- ✅ 现货和合约同时平仓
- ✅ 减少价差变化风险
- ✅ 提高平仓成功率

## 配置说明

在 `application-dev.yml` 中添加：

```yaml
# 套利交易配置
arbitrage:
  trading:
    enabled: false                   # 是否启用自动交易
    trade-amount: 100                # 单笔交易金额（USDT）
    max-positions: 5                 # 最大持仓数量
    open-threshold: 0.005            # 开仓阈值（0.5%）
    close-threshold: 0.002           # 平仓阈值（0.2%）
    order-timeout-seconds: 30        # 订单超时时间（秒）

# OKX API 配置
okx:
  api:
    key: ${OKX_API_KEY}
    secret: ${OKX_SECRET_KEY}
    passphrase: ${OKX_PASSPHRASE}
    simulated: true                  # 是否模拟交易
```

## 注意事项

1. **订单超时处理**：如果订单30秒内未成交，需要取消订单
2. **部分成交处理**：如果只有一个订单成交，需要平掉已成交的订单
3. **网络延迟**：WebSocket 可能有延迟，需要容忍一定的时间差
4. **状态恢复**：Flink 重启后需要重新订阅 WebSocket
5. **资金管理**：确保账户有足够的资金用于交易
