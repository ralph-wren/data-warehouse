## 注意事项

1. **默认关闭自动交易**: 所有配置文件中 `arbitrage.trading.enabled` 默认为 `false`
2. **模拟交易模式**: `okx.api.simulated` 默认为 `true`
3. **API 密钥安全**: 使用环境变量,不要提交到代码仓库
4. **先测试连接**: 使用 `FlinkOrderMonitorJob` 测试 WebSocket 连接
5. **小金额测试**: 实盘测试时使用小金额
6. **监控日志**: 密切关注日志输出,及时发现问题

## 常见问题

### Q1: WebSocket 连接失败
**A**: 
- 检查网络连接
- 确认能够访问 `wss://ws.okx.com:8443`
- 检查防火墙设置

### Q2: 登录认证失败
**A**: 
- 检查 API Key、Secret、Passphrase 是否正确
- 确认环境变量已正确设置
- 检查 API 权限是否包含交易权限

### Q3: 收不到订单消息
**A**: 
- 确保有实际的订单成交
- 可以手动下单测试
- 检查订阅的 instType 是否正确

### Q4: 编译错误
**A**: 
- 确保 Maven 依赖已正确安装
- 检查 Java 版本(需要 Java 11)
- 运行 `mvn clean install`
