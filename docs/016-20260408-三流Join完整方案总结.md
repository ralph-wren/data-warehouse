## 🔧 核心逻辑

### 交易决策流程

```
processElement1(套利机会):
    IF 无持仓:
        IF 价差率 > 开仓阈值:
            → 下单开仓(现货 + 合约)
            → 保存待确认订单
    ELSE IF 有持仓:
        IF 价差回归 OR 超时 OR 止损:
            → 下单平仓(现货 + 合约)
            → 保存待确认订单

processElement2(订单更新):
    IF 订单成交:
        → 更新待确认订单状态
        IF 现货和合约都成交:
            IF 开仓:
                → 确认开仓
                → 更新持仓状态
                → 输出开仓明细
            ELSE IF 平仓:
                → 确认平仓
                → 计算盈亏
                → 更新持仓状态
                → 输出平仓明细
```

### 开仓条件
```java
价差率 > 开仓阈值(0.5%)
AND
无持仓
```

### 平仓条件
```java
(价差回归: 价差率 < 平仓阈值(0.2%))
OR
(超时: 持仓时间 > 最大持仓时间(60分钟))
OR
(止损: 未实现盈亏 < -最大亏损(10 USDT))
```

## 📋 数据库表

### 1. dwd_arbitrage_trades (交易明细表)

**用途**: 记录所有开仓和平仓操作

**字段**:
- symbol: 交易对
- action: OPEN/CLOSE
- direction: LONG_SPOT_SHORT_SWAP / SHORT_SPOT_LONG_SWAP
- amount: 交易金额
- spot_price: 现货价格
- swap_price: 合约价格
- spread_rate: 价差率
- profit: 盈亏(仅平仓时有值)
- close_reason: 平仓原因
- hold_time_ms: 持仓时间
- timestamp: 时间戳

### 2. ads_position_status (持仓状态表)

**用途**: 记录每个币对的当前持仓状态

**字段**:
- symbol: 交易对
- is_open: 是否持仓
- direction: 方向
- amount: 持仓金额
- entry_spot_price: 开仓现货价格
- entry_swap_price: 开仓合约价格
- entry_spread_rate: 开仓价差率
- open_time: 开仓时间
- update_time: 更新时间

### 3. ads_arbitrage_stats (交易统计表)

**用途**: 按币对统计交易表现

**字段**:
- symbol: 交易对
- total_trades: 总交易次数
- win_trades: 盈利次数
- loss_trades: 亏损次数
- win_rate: 胜率
- total_profit: 总盈亏
- avg_profit: 平均盈亏
- max_profit: 最大盈利
- max_loss: 最大亏损
- avg_hold_time_ms: 平均持仓时间
