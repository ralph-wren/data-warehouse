## 📊 实现的功能

### 1. 完整的数据流

```
现货流 (Kafka)  ──┐
                  ├──→ Interval Join ──→ 套利机会流
合约流 (Kafka)  ──┘                          │
                                            │
                                            ↓
                              ┌──────────────────────┐
                              │  套利机会流           │
                              │  ArbitrageOpp        │
                              └──────────────────────┘
                                            │
                                            │ keyBy(symbol)
                                            ↓
订单流 (WebSocket) ──→ ┌────────────────────────────┐
                      │  KeyedCoProcessFunction    │
                      │  TradingDecisionProcessor  │
                      │                            │
                      │  • 判断开仓/平仓/保持       │
                      │  • 触发交易订单            │
                      │  • 确认订单成交            │
                      │  • 维护持仓状态            │
                      └────────────────────────────┘
                                            │
                                            ↓
                              ┌──────────────────────┐
                              │  交易明细流           │
                              │  TradeRecord         │
                              └──────────────────────┘
                                            │
                                            ↓
                              ┌──────────────────────┐
                              │  Doris Sink          │
                              │  dwd_arbitrage_      │
                              │  trades              │
                              └──────────────────────┘
```

### 2. 核心组件

| 组件名称 | 类型 | 功能 |
|---------|------|------|
| OrderUpdate | 数据类 | 订单更新数据结构 |
| PendingOrder | 数据类 | 待确认订单数据结构 |
| OrderUpdateParser | 解析器 | 解析 WebSocket 订单消息 |
| TradingDecisionProcessor | 处理器 | 交易决策核心逻辑 |

### 3. 交易决策逻辑

#### 开仓条件
```java
价差率 > 0.5% AND 无持仓
```

#### 平仓条件
```java
(价差回归: 价差率 < 0.2%)
OR
(超时: 持仓时间 > 60分钟)
OR
(止损: 未实现盈亏 < -10 USDT)
```

#### 订单确认
```java
现货订单成交 AND 合约订单成交 → 确认开仓/平仓
```
