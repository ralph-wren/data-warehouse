# Flink Doris 项目日志和版本兼容性优化

## 问题描述

在运行 Flink ODS 作业时遇到以下问题:

1. **控制台不打印日志**: 运行时看不到详细的日志输出,无法调试问题
2. **写入 Doris 报错**: Stream Load 写入失败,错误信息不清晰
3. **版本兼容性问题**: Flink 1.18 + Doris Connector 24.0.x 存在兼容性问题

## 解决方案

### 1. 日志配置优化

#### 问题分析
- 原有配置只有 logback.xml,但 Flink 运行时默认使用 Log4j2
- 缺少 Log4j2 配置文件导致日志无法正常输出

#### 解决步骤

**步骤 1: 添加 Log4j2 配置文件**

创建 `src/main/resources/log4j2.properties`:

```properties
# 根日志级别和输出器
rootLogger.level = INFO
rootLogger.appenderRef.console.ref = ConsoleAppender
rootLogger.appenderRef.file.ref = FileAppender

# 控制台输出器配置
appender.console.name = ConsoleAppender
appender.console.type = CONSOLE
appender.console.target = SYSTEM_OUT
appender.console.layout.type = PatternLayout
appender.console.layout.pattern = %d{yyyy-MM-dd HH:mm:ss,SSS} %-5p [%t] %c{1}:%L - %m%n

# 项目包日志级别
logger.crypto.name = com.crypto.dw
logger.crypto.level = DEBUG
```

**步骤 2: 添加 Log4j2 依赖**

在 `pom.xml` 中添加:

```xml
<!-- Log4j2 核心依赖 -->
<dependency>
    <groupId>org.apache.logging.log4j</groupId>
    <artifactId>log4j-slf4j-impl</artifactId>
    <version>2.20.0</version>
</dependency>

<dependency>
    <groupId>org.apache.logging.log4j</groupId>
    <artifactId>log4j-api</artifactId>
    <version>2.20.0</version>
</dependency>

<dependency>
    <groupId>org.apache.logging.log4j</groupId>
    <artifactId>log4j-core</artifactId>
    <version>2.20.0</version>
</dependency>
```

### 2. DorisStreamLoadSink 错误处理优化

#### 问题分析
- 原有代码在读取 HTTP 错误响应时会抛出异常
- 日志输出不够详细,无法定位问题
- 响应状态判断逻辑不完善

#### 解决步骤

**优化点 1: 增强日志输出**

```java
logger.info("========== 开始刷新数据到 Doris ==========");
logger.info("记录数: {}", recordCount);
logger.info("Stream Load URL: {}", loadUrl);
logger.info("数据大小: {} bytes", data.length());
```

**优化点 2: 正确处理 HTTP 错误流**

```java
// 根据响应码选择读取正常流或错误流
try (BufferedReader br = new BufferedReader(
        new InputStreamReader(
            responseCode >= 400 ? conn.getErrorStream() : conn.getInputStream(), 
            StandardCharsets.UTF_8))) {
    String line;
    while ((line = br.readLine()) != null) {
        response.append(line);
    }
} catch (Exception e) {
    logger.warn("读取响应流失败: {}", e.getMessage());
}
```

**优化点 3: 完善响应状态判断**

```java
// 解析响应 JSON
JSONObject responseJson = JSONObject.parseObject(responseStr);

if (responseCode == 200 && responseJson != null) {
    String status = responseJson.getString("Status");
    
    if ("Success".equalsIgnoreCase(status)) {
        logger.info("✓ Stream Load 成功: {} 条记录", recordCount);
        logger.info("导入统计: {}", responseJson.toJSONString());
    } else {
        logger.error("✗ Stream Load 失败: status={}", status);
        logger.error("错误信息: {}", responseJson.getString("Message"));
        throw new Exception("Stream Load failed with status: " + status);
    }
}
```

### 3. 版本兼容性优化

#### 问题分析

根据 [Apache Doris Flink Connector 官方文档](https://github.com/apache/doris-flink-connector),不同版本的兼容性如下:

| Flink 版本 | JDK 要求 | 推荐 Connector 版本 | 稳定性 |
|-----------|---------|-------------------|--------|
| 1.17.x | JDK 8 | 1.6.x | ⭐⭐⭐⭐⭐ 最稳定 |
| 1.18.x | JDK 8 | 24.0.x | ⭐⭐⭐ 较新,可能有问题 |
| 1.19.x | JDK 8 | 24.0.x | ⭐⭐ 新版本,不够稳定 |
| 1.20.x | JDK 8 | 24.0.x | ⭐⭐ 新版本,不够稳定 |

**结论**: Flink 1.17.2 + Doris Connector 1.6.2 是最稳定的组合

#### 解决步骤

**修改 pom.xml 版本配置**:

```xml
<properties>
    <!-- 使用稳定的 Flink 1.17 版本,兼容性更好 -->
    <flink.version>1.17.2</flink.version>
    <jackson.version>2.15.2</jackson.version>
    <slf4j.version>1.7.36</slf4j.version>
    <log4j.version>2.20.0</log4j.version>
</properties>

<dependencies>
    <!-- Flink Kafka Connector - 使用与 Flink 1.17 匹配的版本 -->
    <dependency>
        <groupId>org.apache.flink</groupId>
        <artifactId>flink-connector-kafka</artifactId>
        <version>3.0.2-1.17</version>
    </dependency>

    <!-- Flink Connector Base -->
    <dependency>
        <groupId>org.apache.flink</groupId>
        <artifactId>flink-connector-base</artifactId>
        <version>1.17.2</version>
    </dependency>

    <!-- Doris Flink Connector - 使用 1.6.2 版本,更稳定 -->
    <dependency>
        <groupId>org.apache.doris</groupId>
        <artifactId>flink-doris-connector-1.17</artifactId>
        <version>1.6.2</version>
    </dependency>
</dependencies>
```

## 版本选择建议

```
Flink: 1.17.2
JDK: 8 或 11
Doris Connector: 1.6.2
Kafka Connector: 3.0.2-1.17
```

**优势**:
- ✅ 经过长期验证,稳定性最高
- ✅ 社区支持完善,问题解决方案多

- ✅ 性能优秀,资源占用合理
