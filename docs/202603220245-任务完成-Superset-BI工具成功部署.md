## 使用指南

### 启动/停止

```bash
# 启动
bash manage-superset.sh start

# 停止
bash manage-superset.sh stop

# 重启
bash manage-superset.sh restart

# 查看日志
bash manage-superset.sh logs
```

### 首次登录

1. 访问 http://localhost:8088
2. 使用账号 `admin` / 密码 `admin` 登录
3. 建议修改默认密码

### 连接 Doris 数据源

1. 点击右上角 "+" → "Data" → "Connect database"
2. 选择 "Apache Doris" 或 "MySQL"
3. 填写连接信息:
   ```
   Host: doris-fe
   Port: 9030
   Database: crypto_dw
   Username: root
   Password: (留空)
   ```
4. 测试连接并保存

### 创建图表

1. 进入 "SQL Lab"
2. 选择 Doris 数据源
3. 编写 SQL 查询
4. 点击 "Explore" 创建可视化图表
5. 保存到仪表板

## 技术细节

### 当前配置

- **元数据库**: SQLite (容器内,临时)
- **缓存**: 内存缓存
- **镜像**: apache/superset:latest
- **网络**: data-warehouse-net (桥接)

### 尝试过的方案

1. ❌ 使用本地 MySQL - 遇到 pymysql 驱动安装问题
2. ❌ 使用本地 Redis - 配置复杂度高
3. ✅ 使用 Docker PostgreSQL 和 Redis - 简单可靠

### 已知问题

1. 配置文件挂载失败(Git Bash 路径转换)
2. 当前使用 SQLite,重启容器会丢失元数据
3. Redis 缓存未启用

### 后续优化建议

1. 修复配置文件挂载,启用 PostgreSQL
2. 配置 Redis 缓存提升性能
3. 添加数据持久化卷
4. 配置 HTTPS 访问
5. 集成 LDAP/OAuth 认证
