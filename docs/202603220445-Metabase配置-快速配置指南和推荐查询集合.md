## 关键技术点

### 1. Doris 连接配置

```
Database type: MySQL  (Doris 兼容 MySQL 协议)
Host: doris-fe        (Docker 网络内部主机名)
Port: 9030            (Doris MySQL 端口)
Database: crypto_dw   (数据仓库名称)
Username: root
Password: (空)
```

### 2. SQL 查询优化

使用窗口函数获取最新数据:
```sql
SELECT * FROM (
  SELECT 
    *,
    ROW_NUMBER() OVER (PARTITION BY inst_id ORDER BY timestamp DESC) AS rn
  FROM dwd_crypto_ticker_detail
) t
WHERE rn = 1
```

### 3. 时间处理

Doris 中的时间戳是毫秒级,需要转换:
```sql
FROM_UNIXTIME(timestamp / 1000) AS 时间
```

### 4. 时区配置

确保 Metabase 和 Doris 都使用东八区:
- Metabase: Admin settings → Localization → Asia/Shanghai
- Doris: `SET GLOBAL time_zone = 'Asia/Shanghai'`
