# 代码重构完成 - 四个 Flink 作业全部使用工厂模式

**时间**: 2026-03-22 23:00  
**类型**: 代码重构  
**状态**: ✅ 完成

## 问题描述

四个 Flink 作业存在大量重复代码:
- Environment 创建逻辑重复(Web UI、Metrics、Checkpoint 配置)
- Table 创建逻辑重复(Kafka Source、Doris Source/Sink DDL)
- 字段定义分散在各个作业中
- 端口配置容易冲突
- 维护成本高

## 解决方案

### 1. 创建三个工厂类

#### FlinkEnvironmentFactory (Environment 工厂)
- 封装 StreamExecutionEnvironment 创建逻辑
- 封装 StreamTableEnvironment 创建逻辑
- 统一配置 Web UI、Prometheus Metrics、Checkpoint
- 自动设置时区为 Asia/Shanghai (东八区)
- 支持自定义端口避免冲突

#### FlinkTableFactory (Table 工厂)
- 封装 Kafka Source 表创建逻辑
- 封装 Doris Source 表创建逻辑
- 封装 Doris Sink 表创建逻辑
- 支持可选的 Watermark 配置

#### TableSchemas (Schema 定义)
- 集中管理所有表的字段定义
- 包含 Kafka Ticker、Doris ODS、Doris DWD、Doris DWS 字段
- 统一字段类型和注释

### 2. 重构四个 Flink 作业

#### FlinkODSJobSQL (ODS 层 - SQL)
- **重构前**: 约 200 行
- **重构后**: 约 100 行
- **减少**: 50% (约 100 行)
- **使用**: FlinkEnvironmentFactory.createTableEnvironment() + FlinkTableFactory
- **端口**: 8081

#### FlinkODSJobDataStream (ODS 层 - DataStream)
- **重构前**: 约 220 行
- **重构后**: 约 160 行
- **减少**: 27% (约 60 行)
- **使用**: FlinkEnvironmentFactory.createStreamEnvironment()
- **端口**: 8083
- **说明**: DataStream API 保留了更多自定义逻辑(Kafka Source、Doris Sink)

#### FlinkDWDJobSQL (DWD 层 - SQL)
- **重构前**: 280 行
- **重构后**: 110 行
- **减少**: 61% (约 170 行)
- **使用**: FlinkEnvironmentFactory.createTableEnvironment() + FlinkTableFactory
- **端口**: 8082

#### FlinkDWSJob1MinSQL (DWS 层 - SQL)

## 使用示例

### Table API 作业

```java
// 加载配置
ConfigLoader config = ConfigLoader.getInstance();

// 创建 Table Environment (自动配置 Web UI、Metrics、Checkpoint、时区)
FlinkEnvironmentFactory envFactory = new FlinkEnvironmentFactory(config);
StreamTableEnvironment tableEnv = envFactory.createTableEnvironment("job-name", 8081);

// 创建表
FlinkTableFactory tableFactory = new FlinkTableFactory(config);

// Kafka Source 表
String kafkaSourceDDL = tableFactory.createKafkaSourceTable(
    "kafka_source",
    TableSchemas.KAFKA_TICKER_SCHEMA,
    true  // 需要 Watermark
);
tableEnv.executeSql(kafkaSourceDDL);

// Doris Sink 表
String dorisSinkDDL = tableFactory.createDorisSinkTable(
    "doris_sink",
    "ods",  // 表类型: ods/dwd/dws
    TableSchemas.DORIS_ODS_SOURCE_SCHEMA
);
tableEnv.executeSql(dorisSinkDDL);

// 执行 SQL
tableEnv.executeSql("INSERT INTO doris_sink SELECT * FROM kafka_source");
```

### DataStream API 作业

```java
// 加载配置
ConfigLoader config = ConfigLoader.getInstance();

// 创建 Stream Environment (自动配置 Web UI、Metrics、Checkpoint)
FlinkEnvironmentFactory envFactory = new FlinkEnvironmentFactory(config);
StreamExecutionEnvironment env = envFactory.createStreamEnvironment("job-name", 8083);

// 创建 Kafka Source (自定义)
KafkaSource<String> kafkaSource = KafkaSource.<String>builder()
    .setBootstrapServers(config.getString("kafka.bootstrap-servers"))
    .setTopics(config.getString("kafka.topic.crypto-ticker"))
    .setGroupId("consumer-group")
    .setStartingOffsets(OffsetsInitializer.latest())
    .setValueOnlyDeserializer(new SimpleStringSchema())
    .build();

// 创建数据流
DataStream<String> stream = env.fromSource(
    kafkaSource,
    WatermarkStrategyFactory.forBoundedOutOfOrdernessString(5),
    "Kafka Source"
);

// 执行作业
env.execute("Job Name");
```
