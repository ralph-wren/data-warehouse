### 3.2 缺少配置验证

**位置**: `ConfigLoader.java`

**问题**: 启动时不验证必需配置项，运行时才发现配置缺失

**影响**: 
- 作业启动后才失败，浪费时间
- 错误信息不明确

**优化方案**: 添加配置验证方法

```java
public class ConfigLoader {
    
    /**
     * 验证必需的配置项
     */
    public void validateRequiredConfigs() {
        String[] requiredKeys = {
            "kafka.bootstrap-servers",
            "kafka.topic.crypto-ticker",
            "doris.fe.http-url",
            "doris.database"
        };
        
        List<String> missingKeys = new ArrayList<>();
        for (String key : requiredKeys) {
            if (getString(key) == null || getString(key).isEmpty()) {
                missingKeys.add(key);
            }
        }
        
        if (!missingKeys.isEmpty()) {
            throw new IllegalStateException(
                "Missing required configuration keys: " + String.join(", ", missingKeys)
            );
        }
        
        logger.info("✓ All required configurations are present");
    }
}
```

**位置**: `ConfigLoader.loadYamlFile()`

**问题**:

- 大量调试日志，影响性能

**影响**:

- 代码可读性差
- 维护成本高

**优化方案**: 简化加载逻辑，减少调试日志

```java
private Map<String, Object> loadYamlFile(String filename) {
    try {
        Yaml yaml = new Yaml();
        
        // 尝试标准路径
        String[] paths = {
            "config/" + filename,
            filename
        };
        
        for (String path : paths) {
            InputStream inputStream = getClass().getClassLoader().getResourceAsStream(path);
            if (inputStream != null) {
                logger.info("Loading config from: {}", path);
                Map<String, Object> data = yaml.load(inputStream);
                inputStream.close();
                return data != null ? data : new HashMap<>();
            }
        }
        
        logger.error("Config file not found: {}", filename);
        return new HashMap<>();
        
    } catch (Exception e) {
        logger.error("Error loading config: {}", filename, e);
        return new HashMap<>();
    }
}
```

public class DataValidationUtils {
    
    /**
     * 验证价格是否合法
     */
    public static boolean isValidPrice(BigDecimal price) {
        return price != null && 
               price.compareTo(BigDecimal.ZERO) > 0 && 
               price.compareTo(new BigDecimal("10000000")) < 0;
    }
    
    /**
     * 验证时间戳是否合法
     */
    public static boolean isValidTimestamp(long timestamp) {
        long now = System.currentTimeMillis();
        long tolerance = 3600000L; // 1 小时
        return Math.abs(now - timestamp) < tolerance;
    }
}
