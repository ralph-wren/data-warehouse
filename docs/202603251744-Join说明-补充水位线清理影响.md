# Interval Join 水位线清理影响补充说明

## 目标

补充 Interval Join 中水位线对状态清理时机的影响，避免理解为“固定等待时间”。

## 处理步骤

1. 在 `FlinkADSArbitrageJob` 注释中增加水位线影响清理时机的说明。
2. 用简短条目强调“过期由水位线推进决定”。

## 关键说明

- **状态清理时机**：数据是否过期取决于水位线推进到上界，而不是固定等待时长。
- **实际含义**：即使事件时间差超过 ±10 秒，只要水位线未推进到上界，仍可能保留在状态中等待匹配。

## 结果

已补充水位线影响清理时机的说明，帮助理解 Interval Join 的“等待”机制。
