## 核心规范要点

### 1. 命名规范

**类名**: 大驼峰命名法
```java
FlinkEnvironmentFactory  // ✅ 正确
flinkEnvironmentFactory  // ❌ 错误
```

**方法名**: 小驼峰命名法
```java
createStreamEnvironment()  // ✅ 正确
CreateStreamEnvironment()  // ❌ 错误
```

**SQL 文件**: 小写字母，下划线分隔
```
ods_insert.sql           // ✅ 正确
ODS_Insert.sql           // ❌ 错误
odsInsert.sql            // ❌ 错误
```

### 2. 注释规范

**类注释**: 必须包含 JavaDoc
```java
/**
 * Flink Environment 工厂类
 * 
 * 封装 Flink 执行环境的创建逻辑,避免重复代码
 * 支持的功能:
 * - DataStream API 环境
 * - Table API 环境
 * - Web UI 配置
 * 
 * @author Kiro AI
 * @date 2026-03-25
 */
public class FlinkEnvironmentFactory {
    // ...
}
```

**方法注释**: 公共方法必须包含 JavaDoc
```java
/**
 * 创建 StreamExecutionEnvironment (DataStream API)
 * 
 * @param jobName 作业名称(用于 Metrics 标识)
 * @param webUIPort Web UI 端口(避免端口冲突)
 * @return 配置好的 StreamExecutionEnvironment
 */
public StreamExecutionEnvironment createStreamEnvironment(String jobName, int webUIPort) {
    // ...
}
```

### 3. SQL 分离规范

**禁止硬编码**:
```java
// ❌ 错误：硬编码 SQL
private static String createInsertSQL() {
    return "INSERT INTO doris_ods_sink\n" +
           "SELECT \n" +
           "    inst_id,\n" +
           // ...
}

// ✅ 正确：从文件加载
String sql = SqlFileLoader.loadSql("sql/flink/ods_insert.sql");
```

**SQL 文件组织**:
```
sql/
├── flink/              # Flink SQL
│   ├── ddl/            # DDL 定义
│   ├── ods_insert.sql  # ODS 层
│   ├── dwd_insert.sql  # DWD 层
│   ├── dws_1min_insert.sql  # DWS 层
│   └── ads_realtime_metrics_insert.sql  # ADS 层
└── doris/              # Doris 建表 SQL
```

### 4. 工厂模式规范

**必须使用工厂类**:
```java
// ✅ 正确：使用工厂类
FlinkEnvironmentFactory factory = new FlinkEnvironmentFactory(config);
StreamExecutionEnvironment env = factory.createStreamEnvironment("job-name", 8081);

// ❌ 错误：直接创建环境
StreamExecutionEnvironment env = StreamExecutionEnvironment.getExecutionEnvironment();
```

**工厂类列表**:
- `FlinkEnvironmentFactory`: 创建 Flink 执行环境
- `FlinkTableFactory`: 创建 Flink 表（Source/Sink）
- `KafkaSourceFactory`: 创建 Kafka Source
- `DorisSinkFactory`: 创建 Doris Sink

### 5. 配置管理规范

**环境配置文件**:
```
config/
├── application-dev.yml      # 本地开发环境（宿主机地址）
└── application-docker.yml   # Docker 容器环境（容器名称）
```

**环境切换**:
```bash
# 本地模式
java -jar app.jar --APP_ENV dev

# 远程集群模式
java -jar app.jar --APP_ENV docker
```

**禁止事项**:
- ❌ 禁止在配置文件中嵌套环境配置
- ❌ 禁止在代码中根据环境判断配置

**问题解决文档命名**:

### 7. Git 提交规范

**提交信息格式**:
```
<type>(<scope>): <subject>

<body>

<footer>
```

**Type 类型**:
- `feat`: 新功能
- `fix`: 修复 bug
- `refactor`: 重构代码
- `docs`: 文档更新
- `style`: 代码格式调整
- `test`: 测试相关
- `chore`: 构建/工具相关

**示例**:
```

- 创建 ods_insert.sql 和 ads_realtime_metrics_insert.sql
- 修改 FlinkODSJobSQL 和 FlinkADSRealtimeMetricsJob 使用 SqlFileLoader
- 更新 SQL README 文档

Closes #72
```
