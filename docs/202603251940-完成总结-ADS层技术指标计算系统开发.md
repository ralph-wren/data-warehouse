#### 自定义指标示例
1. **成交量加权移动平均线（VWMA）**
   - 文件: `src/main/java/com/crypto/dw/indicators/CustomVolumeWeightedMA.java`
   - 公式: VWMA = Σ(价格 × 成交量) / Σ(成交量)
   - 优势: 考虑成交量影响，更真实反映市场

2. **价格通道指标（Price Channel）**
   - 文件: `src/main/java/com/crypto/dw/indicators/CustomPriceChannelIndicator.java`
   - 输出: 上轨、中轨、下轨、通道宽度、价格位置
   - 用途: 判断突破和支撑/阻力位

## 技术亮点

### 1. TA4J集成 ⭐⭐⭐

**优势**:
- 100+种内置技术指标
- 高精度数值计算（Num类型）
- 完整的K线数据结构
- 支持自定义指标

**使用示例**:
```java
// 创建K线序列
BarSeries series = new BaseBarSeriesBuilder()
    .withName("BTC-USDT")
    .withNumTypeOf(DecimalNum.class)
    .build();

// 计算MA指标
ClosePriceIndicator closePrice = new ClosePriceIndicator(series);
SMAIndicator sma = new SMAIndicator(closePrice, 20);
Num value = sma.getValue(series.getEndIndex());
```

### 2. 动态配置管理 ⭐⭐⭐

**实现方式**:
- 配置存储在Redis中
- 使用广播状态分发配置
- 每30秒自动刷新
- 无需重启作业

**配置格式**:
```json
{
    "config_id": "ma_20",
    "symbol": "*",
    "indicator_type": "MA",
    "indicator_name": "MA20",
    "params": {"period": "20"},
    "enabled": true
}
```

### 3. 自定义指标扩展 ⭐⭐⭐

**接口定义**:
```java
public interface CustomIndicator {
    void calculate(
        BarSeries series,
        Map<String, String> params,
        IndicatorResult result
    );
}
```

**加载机制**:
- 通过反射动态加载
- 支持任意自定义逻辑
- 与内置指标无缝集成

### 4. 状态管理 ⭐⭐

**K线序列状态**:
- 使用MapState存储
- 每个交易对独立维护
- 限制最大500根K线
- 状态持久化到RocksDB

**优势**:
- 支持故障恢复
- 内存可控
- 增量计算

## 使用指南

### 快速开始

```bash
# 1. 初始化Redis配置
./scripts/init-technical-indicators.sh

# 2. 创建Doris表
mysql -h 127.0.0.1 -P 9030 -u root < sql/doris/create_ads_technical_indicators_table.sql

# 3. 编译打包
mvn clean package -DskipTests

# 4. 测试验证
./test/test-technical-indicators-job.sh

# 5. 启动作业
./run-flink-ads-technical-indicators.sh
```

### 动态添加指标

```bash
# 添加MA30指标
redis-cli SET "indicator:config:ma_30" '{
    "config_id": "ma_30",
    "symbol": "*",
    "indicator_type": "MA",
    "indicator_name": "MA30",
    "params": {"period": "30"},
    "enabled": true,
    "update_time": '$(date +%s)000'
}'

# 作业会在30秒内自动加载新配置
```

### 查询指标数据

```sql
-- 查看最新指标
SELECT 
    symbol, indicator_name, indicator_value, create_time
FROM crypto_ads.ads_technical_indicators
ORDER BY timestamp DESC LIMIT 10;

-- 查看MACD指标
SELECT 
    symbol,
    indicator_value as macd_bar,
    JSON_EXTRACT(extra_values, '$.dif') as dif,
    JSON_EXTRACT(extra_values, '$.dea') as dea
FROM crypto_ads.ads_technical_indicators
WHERE indicator_type = 'MACD'
ORDER BY timestamp DESC LIMIT 10;
```
