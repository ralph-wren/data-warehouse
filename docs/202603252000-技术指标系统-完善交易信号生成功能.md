#### 2.3 完善各指标的信号生成规则

为每个指标添加详细的交易信号规则说明：

**MA（移动平均线）**：
- BUY：价格上穿MA（金叉），信号强度70
- SELL：价格下穿MA（死叉），信号强度70
- HOLD：价格在MA上方或下方，信号强度30

**EMA（指数移动平均线）**：
- BUY：价格上穿EMA（金叉），信号强度75
- SELL：价格下穿EMA（死叉），信号强度75
- HOLD：价格在EMA上方或下方，信号强度30

**RSI（相对强弱指标）**：
- BUY：RSI < 30（超卖），信号强度 = 100 - RSI×2
- SELL：RSI > 70（超买），信号强度 = RSI - 30
- HOLD：30 ≤ RSI ≤ 70（中性），信号强度20

**MACD**：
- BUY：DIF上穿DEA（金叉），信号强度 = min(|MACD柱|×100, 90)
- SELL：DIF下穿DEA（死叉），信号强度 = min(|MACD柱|×100, 90)
- HOLD：DIF在DEA上方或下方，信号强度40

**BOLL（布林带）**：
- BUY：价格接近下轨（position < 10%），信号强度 = (10 - position) × 8
- SELL：价格接近上轨（position > 90%），信号强度 = (position - 90) × 8
- HOLD：价格在中轨上方或下方，信号强度30

**KDJ**：
- BUY：K上穿D金叉且K < 50，信号强度85
- BUY：K < 20（超卖），信号强度 = (20 - K) × 4
- SELL：K下穿D死叉且K > 50，信号强度85
- SELL：K > 80（超买），信号强度 = (K - 80) × 4
- HOLD：20 ≤ K ≤ 80，信号强度30

**ATR（平均真实波幅）**：
- 不生成交易信号，仅用于风险管理
- 信号类型：HOLD

**CUSTOM（自定义指标）**：
- 由自定义指标实现决定，如果未实现则默认HOLD

#### 2.4 添加交易信号查询示例

```sql
-- 查看最新的交易信号
SELECT 
    symbol,
    indicator_name,
    signal_type,
    signal_reason,
    signal_strength,
    current_price,
    create_time
FROM crypto_ads.ads_technical_indicators
WHERE signal_type IN ('BUY', 'SELL')
ORDER BY timestamp DESC
LIMIT 20;

-- 查看强信号（强度>70）
SELECT 
    symbol,
    indicator_name,
    signal_type,
    signal_strength,
    current_price
FROM crypto_ads.ads_technical_indicators
WHERE signal_strength > 70
ORDER BY timestamp DESC;

-- 统计各类信号数量
SELECT 
    signal_type,
    COUNT(*) as signal_count,
    AVG(signal_strength) as avg_strength
FROM crypto_ads.ads_technical_indicators
WHERE create_time >= DATE_SUB(NOW(), INTERVAL 1 HOUR)
GROUP BY signal_type;
```

## 技术要点

### 1. 交易信号生成规则设计

每个指标都有自己的信号生成规则，主要分为三类：

**趋势类指标（MA、EMA、MACD）**：
- 基于金叉死叉判断
- 金叉（上穿）→ BUY
- 死叉（下穿）→ SELL

**震荡类指标（RSI、KDJ）**：
- 基于超买超卖区域判断
- 超卖区域（低位）→ BUY
- 超买区域（高位）→ SELL

**通道类指标（BOLL）**：
- 基于价格在通道中的位置判断
- 接近下轨 → BUY
- 接近上轨 → SELL

### 2. 信号强度计算

信号强度范围：0-100

**固定强度**：
- 金叉死叉：70-85（根据指标类型）
- 中性持有：20-40

**动态强度**：
- RSI：根据偏离中性区域的程度计算
- MACD：根据MACD柱的大小计算
- BOLL：根据价格在通道中的位置计算
- KDJ：根据K值偏离中性区域的程度计算

### 3. 信号原因说明

每个信号都包含详细的原因说明，例如：
- "Price crosses above MA20"（价格上穿MA20）
- "RSI oversold (< 30)"（RSI超卖）
- "MACD golden cross (DIF > DEA)"（MACD金叉）
- "Price near lower Bollinger Band"（价格接近布林带下轨）

这些说明帮助用户理解信号产生的原因。

### 4. 数据结构设计

**IndicatorResult实体**包含完整的信号信息：

## 使用示例

```sql

    indicator_name,
    signal_reason,
    signal_strength,
    current_price,
    create_time

WHERE signal_type = 'BUY'
```
