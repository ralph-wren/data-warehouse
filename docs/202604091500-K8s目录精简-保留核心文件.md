## 使用建议

### 快速部署
```bash
# 1. 创建共享资源
kubectl apply -f k8s/common/shared-resources.yaml

# 2. 拉取镜像
docker pull flink:1.17.2-java11

# 3. 部署作业
cd k8s
./deploy-job.sh deploy ticker-collector
```

### 查看文档
```bash
# 查看部署指南
cat k8s/README.md

# 查看问题修复记录
cat k8s/TROUBLESHOOTING.md
```

### 管理作业
```bash
# 查看所有作业
./deploy-job.sh list

# 查看作业状态
./deploy-job.sh status ticker-collector

# 查看作业日志
./deploy-job.sh logs ticker-collector
```
