# 借币利息和手续费功能实现

## 问题描述

在套利交易中,需要考虑两个重要的成本因素:

1. **借币利息**: 做空现货时,如果持仓不足需要借币,会产生借币利息成本
2. **交易手续费**: 现货和合约交易都有手续费,需要从净收益中扣除

之前的套利计算没有考虑这些成本,导致净收益计算不准确。

## 手续费标准

根据OKX交易所的费率:
- 现货手续费: 0.1% (千分之一)
- 合约手续费: 0.05% (万分之五)

在 `PositionState` 类中添加两个字段:

/**
 * 开仓手续费（USDT）
 * 包含现货和合约的开仓手续费总和

### 4. 开仓时保存成本信息

在 `confirmOpen()` 方法中:

1. 计算并保存开仓手续费(现货 + 合约)
2. 如果是做空现货策略,保存借币利息率

```java
// 计算开仓手续费
BigDecimal openFeeUsdt = BigDecimal.ZERO;
if (pending.spotFee != null) {
    BigDecimal spotFeeUsdt = pending.spotFee.abs().multiply(pending.spotFillPrice);
    openFeeUsdt = openFeeUsdt.add(spotFeeUsdt);
}
if (pending.swapFee != null) {
    BigDecimal swapFeeUsdt = pending.swapFee.abs();
    openFeeUsdt = openFeeUsdt.add(swapFeeUsdt);
}

// 如果是做空现货,保存借币利息率
if (direction.equals("SHORT_SPOT_LONG_SWAP")) {
    BigDecimal borrowRate = tradingService.getLastBorrowInterestRate();
    if (borrowRate != null) {
        position.setBorrowInterestRate(borrowRate);
    }
}

position.setOpenFeeUsdt(openFeeUsdt);
```

### 5. 平仓时计算净收益

在 `confirmClose()` 方法中,按顺序扣除各项成本:

```java
// 1. 计算原始收益
BigDecimal profit = spreadChange.multiply(position.getAmount());

// 2. 扣除开仓手续费
if (position.getOpenFeeUsdt() != null) {
    profit = profit.subtract(position.getOpenFeeUsdt());
}

// 3. 扣除平仓手续费
BigDecimal closeFeeUsdt = BigDecimal.ZERO;
if (pending.spotFee != null) {
    closeFeeUsdt = closeFeeUsdt.add(pending.spotFee.abs().multiply(pending.spotFillPrice));
}
if (pending.swapFee != null) {
    closeFeeUsdt = closeFeeUsdt.add(pending.swapFee.abs());
}
profit = profit.subtract(closeFeeUsdt);

// 4. 如果是做空现货,扣除借币利息
if (position.getDirection().equals("SHORT_SPOT_LONG_SWAP")) {
    BigDecimal borrowRate = position.getBorrowInterestRate();
    if (borrowRate != null && borrowRate.compareTo(BigDecimal.ZERO) > 0) {
        int holdTimeHours = (int) Math.ceil(holdTimeSeconds / 3600.0);
        BigDecimal borrowInterestCost = position.getAmount()
            .multiply(borrowRate)
            .multiply(new BigDecimal(holdTimeHours));
        BigDecimal borrowInterestCostUsdt = borrowInterestCost.multiply(pending.spotFillPrice);
        profit = profit.subtract(borrowInterestCostUsdt);
    }
}
```

## 计算公式

### 手续费计算

```
现货手续费(USDT) = 现货手续费(币数量) × 成交价格
合约手续费(USDT) = 合约手续费(已经是USDT)
总手续费 = 开仓现货手续费 + 开仓合约手续费 + 平仓现货手续费 + 平仓合约手续费
```

### 借币利息计算

```
利息（币数量） = 借币数量 × 小时利率 × 持仓小时数
利息（USDT） = 利息（币数量） × 平仓价格
```

### 净收益计算

```
原始收益 = (平仓价差 - 开仓价差) × 持仓数量
净收益 = 原始收益 - 开仓手续费 - 平仓手续费 - 借币利息成本
```

## 示例

假设做空现货套利:
- 交易金额: 1000 USDT
- 开仓价格: BTC 50000 USDT
- 平仓价格: BTC 49000 USDT
- 持仓时间: 24 小时
- 借币利率: 0.0000040833333334 (小时)

计算过程:

1. **交易数量**: 1000 / 50000 = 0.02 BTC

2. **手续费**:
   - 开仓现货手续费: 0.02 × 50000 × 0.001 = 1 USDT
   - 开仓合约手续费: 1000 × 0.0005 = 0.5 USDT
   - 平仓现货手续费: 0.02 × 49000 × 0.001 = 0.98 USDT
   - 平仓合约手续费: 980 × 0.0005 = 0.49 USDT
   - 总手续费: 1 + 0.5 + 0.98 + 0.49 = 2.97 USDT

3. **借币利息**:
   - 利息(BTC): 0.02 × 0.0000040833333334 × 24 = 0.00000196 BTC
   - 利息(USDT): 0.00000196 × 49000 = 0.096 USDT

4. **净收益**:
   - 原始收益: (50000 - 49000) × 0.02 = 20 USDT
   - 净收益: 20 - 2.97 - 0.096 = 16.93 USDT

## 注意事项

1. OKX返回的手续费是负数,表示扣除的金额,需要取绝对值
2. 现货手续费币种可能是交易币种,需要转换为USDT
3. 合约手续费通常直接是USDT
4. 借币利息按小时计算,持仓时间向上取整到小时
5. 只有做空现货时才需要查询和计算借币利息
