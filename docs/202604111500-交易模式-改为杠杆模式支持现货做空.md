# 交易模式改造 - 改为杠杆模式支持现货做空

## 问题描述

现货卖出(做空)失败,错误信息:
```
Order failed. Your available BAND balance is insufficient, and your available margin (in USD) is too low for borrowing.
```

**根本原因**:
- OKX 现货卖出(做空)需要借币,必须使用杠杆模式(`tdMode: "cross"` 或 `"isolated"`)
- 之前代码使用 `tdMode: "cash"` 模式,只能卖出已持有的币
- 策略 B (做空现货 + 做多合约) 无法在 `cash` 模式下执行

## 解决方案

### 1. 修改现货交易模式

将 `OKXTradingService` 中的现货交易模式从 `cash` 改为 `cross` (全仓杠杆):

**修改文件**: `src/main/java/com/crypto/dw/trading/OKXTradingService.java`

```java
// buySpot 方法
orderRequest.put("tdMode", "cross");  // 全仓杠杆模式（支持借币做空）

// sellSpot 方法
orderRequest.put("tdMode", "cross");  // 全仓杠杆模式（支持借币做空）
```

### 2. 恢复双向套利策略

恢复 `FlinkADSArbitrageJob` 中的策略 B (做空现货 + 做多合约):

**修改文件**: `src/main/java/com/crypto/dw/flink/FlinkADSArbitrageJob.java`

#### 2.1 套利计算器 - 恢复双向判断

```java
// 判断套利方向
if (spread.compareTo(BigDecimal.ZERO) > 0) {
    // 合约价格 > 现货价格：做空合约，做多现货（策略 A）
    opportunity.arbitrageDirection = "做多现货/做空合约";
} else {
    // 现货价格 > 合约价格：做多合约，做空现货（策略 B）
    opportunity.arbitrageDirection = "做空现货/做多合约";
}
```

#### 2.2 开仓逻辑 - 恢复策略 B

```java
if (opp.arbitrageDirection.contains("做多现货")) {
    // 策略 A: 做多现货 + 做空合约
    direction = "LONG_SPOT_SHORT_SWAP";
    spotOrderId = tradingService.buySpot(opp.symbol, contractSize);
    if (spotOrderId != null) {
        swapOrderId = tradingService.shortSwap(opp.symbol, contractSize, 1);
    }
} else {
    // 策略 B: 做空现货 + 做多合约（使用杠杆模式）
    direction = "SHORT_SPOT_LONG_SWAP";
    spotOrderId = tradingService.sellSpot(opp.symbol, contractSize);
    if (spotOrderId != null) {
        swapOrderId = tradingService.longSwap(opp.symbol, contractSize, 1);
    }
}
```

## 杠杆模式说明

### tdMode 参数说明

- `cash`: 非保证金模式,只能交易已持有的币
- `cross`: 全仓杠杆模式,可以借币交易,所有仓位共享保证金
- `isolated`: 逐仓杠杆模式,每个仓位独立保证金

### 选择 cross 模式的原因

1. **支持借币**: 可以做空现货(卖出未持有的币)
2. **资金利用率高**: 所有仓位共享保证金,不需要为每个仓位单独分配资金
3. **风险可控**: 套利策略是对冲的,现货和合约方向相反,整体风险较低

### 风险提示

1. **借币利息**: 做空现货需要借币,会产生利息费用
2. **保证金要求**: 需要确保账户有足够的保证金,否则可能被强平
3. **杠杆风险**: 虽然是对冲策略,但仍需注意保证金充足

## 测试建议

1. **小额测试**: 先用小额资金测试策略 B 是否能正常执行
2. **监控保证金**: 实时监控账户保证金率,避免强平风险
3. **计算成本**: 对比借币利息和套利收益,确保有利可图

## 总结

通过改为杠杆模式,现在支持:
- 策略 A: 做多现货 + 做空合约 (合约价格 > 现货价格)
- 策略 B: 做空现货 + 做多合约 (现货价格 > 合约价格)

两种策略都可以正常执行,充分利用双向套利机会。
