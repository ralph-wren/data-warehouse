## 统一口径建议（最终版）

建议全项目固定为：

- **价差（绝对值）**：`spread_abs = |swap - spot|`
- **价差率（百分比）**：`spread_rate_pct = spread_abs / min(spot, swap) × 100`

原因：

1. 同时适用于高价差和低价差币种
2. 对上下行方向对称，不偏向某一腿
3. 与“套利机会强度”语义一致（看偏离幅度，不看方向符号）

方向仍由价格关系单独判定：

- `swap > spot`：做多现货 / 做空合约
- `spot > swap`：做空现货 / 做多合约
