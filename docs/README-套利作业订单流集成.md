## 核心组件

### 1. OKXOrderWebSocketSource
**位置**: `data-warehouse/src/main/java/com/crypto/dw/trading/OKXOrderWebSocketSource.java`

**功能**: 订阅 OKX 订单更新,实时接收订单状态

### 2. OrderMatchProcessor
**位置**: `data-warehouse/src/main/java/com/crypto/dw/trading/OrderMatchProcessor.java`

**功能**: 匹配现货和合约订单,确认开仓成功

### 3. OKXTradingService
**位置**: `data-warehouse/src/main/java/com/crypto/dw/trading/OKXTradingService.java`

**功能**: 封装 OKX 交易 API,支持现货和合约交易

### 4. PositionState
**位置**: `data-warehouse/src/main/java/com/crypto/dw/trading/PositionState.java`

**功能**: 持仓状态数据结构

## 注意事项

1. **安全第一**: 不要提交 API 密钥到代码仓库
2. **先测试**: 使用模拟交易模式测试
3. **小金额**: 实盘测试时使用小金额
4. **监控日志**: 密切关注日志输出
5. **设置止损**: 设置最大亏损限制
